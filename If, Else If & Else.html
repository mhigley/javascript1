If, Else If & Else

The if statement  executes a statement if a specified condition is truthy. If the condition is falsy, another statement can be executed.

if()

It's not uncommon to have something specific run only if a certain expression is true, otherwise have nothing happen

var getTime = new Date().getHours ();
if (getTime < 12) {
alert("Good Morning!") ;
}

if(), else

However, having an alternative block of code run should the initial value prove false is a much better option.

var getTime = new Date().getHours ();
if (getTime < 12) {
alert("Good Morning!") ;
}else{
alert("Good Day.") ;
}

if(), else if(), else

If you need an even greater amount of detail, you can break the expression up into an infinite number of parts. It will scan through each one looking for the first expression to be found true and then run the code.

var getTime = new Date().getHours ();
if (getTime < 12) {
alert("Good Morning!") ;
}else if ( getTime < 18){
alert("Good Afternoon.") ;
}else{
alert("Good Evening") ;
}
