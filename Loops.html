Loops

The for statement creates a loop that consists of three optional expressions, enclosed in parentheses and separated by semicolons, followed by a statement (usually a block statement) to be executed in the loop.

Loops are handy, if you want to run the same code over and over again, each time with a different value.

for loop

A long time classic. It starts with the for keyword, then inside the parenthesis has three parameters: an initialization, a condition and a final-expression.

for(var i = 0; i < 5 ; i++){
console.log (i);
}

while loop

The while statement creates a loop that executes a specified statement as long as the test condition evaluates to true.

var i = 0 ;
while(i < 10){
console.log (i);
i++;
}

Tip: Watch Out! Don't forget to add the incrementor inside of your while loop, or else you'll create an infinite loop and crash your browser!!

If you feel as if the for loop and the while loop are very similar... you're right! They are. In the while loop, you've just removed the initialization and final-expression to outside the loop and after the code block respectively. They are essentially identical.

do... while loop

The do... while loop is a variant of the while loop. This loop will execute the code block once, before checking if the condition is true, then it will repeat the loop as long as the condition is true. It will always run at least once!

var i = 0 ;
do {
console.log (i);
i++;
}
while (i < 10);

.forEach() loop

The array specific loop. The forEach() method calls a provided function once for each element in an array, in order.

var jeeps = [ 'wrangler', 'cherokee', 'grand cherokee', 'compass', 'renegade', 'patriot x' ];
jeeps.forEach( function(item, index) {
console.log (item);
});

for... in loop

The JavaScript for/in statement loops through the properties of an object

var jeep = {
make: 'jeep',
model: 'wrangler',
year: 2014,
color: 'black',
doors: 2
};
for(prop in jeep ){
console.log (prop);
}

To access the values of each of the objects properties, pass the object name, and using bracket notation, pass the property name.

for(prop in jeep ){
console.log (jeep[prop]);
}
