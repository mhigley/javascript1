Created in 1995 by Brendan Eich of Netscape in 10 days.
Started out named Mocha, changed to LiveScript and again to JavaScript.
The rendering technology is embedded into every web browser.
Developed so programmers could make engaging and dynamic websites that run in the browser, not on the server.

So how does it work?

There are three ways to add JavaScript to your webpages.

Inline

Generally disuaded technique, however appropriate under certain circumstances

<button type="button" onclick="alert('You created a popup window')">Click Here</button>

Internal

A better solution than inline. Perfect for small scripts that only need to run on a specific page. Blocks of JavaScript can be added throughout your pages in both the <head> and the <body>.

<script>
document.innerHTML = 'Printing some text to the screen';
</script>

Tip: Be careful though... JavaScript will run as soon as the browser comes across it rendering the page!!

External

Ideally, the best solution. Complex scripts get created in a separate file. The <script> can be included in either the <head> or the <body>, but it is generally recommended to be added before the closing <body> tag.

...
<script src="js/script.js"></script>
