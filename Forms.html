Forms

Forms are a pretty complex beast. Accessing the user content can be potentially complicated and done in a number of ways. We'll look at a simple form and a complex form.

Simple form

<!-- HTML -->
<form id="simple">
<input type="text" placeholder="username" id="username" ><br>
<input type="password" placeholder="password" id="password" ><br>
<input type="submit" value="Go!" />
</form>
/* JavaScript */
var sForm = document .getElementById('simple');
sForm.lastElementChild. addEventListener('click', function( e){
e.preventDefault( );
if(sForm. username.value){
console.log ('Welcome ' + sForm.username .value);
}
if(sForm. password.value){
console.log ('Your password ' + sForm.password .value);
}
});
