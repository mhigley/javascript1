The Console


This is where we will do all of our debugging. We will use this a lot! If an error is "thrown", you'll see it here. To access the console, right click anywhere on the browser window and select Inspect element then select the Console tab.

To print a message to the console, in your script.js file, type:

console.log ('Sending a message to the console.')

Note the syntax. console is what's referred to as an object of the window. This object has a many methods for reporting information to the console.

.log()

You've already tried this one out. It displays a simple message in the console and includes a link to the source of the message.

console.log ('Sending a message to the console.')

.info()

Displays a message with an information icon and includes a link to the source of the message.

console.info ('Sending an information message to the console.');

.warn()

Displays a message with a warning icon and includes a link to the source of the message.

console.warn ('Sending a warning message to the console.');

.error()

Displays a message with an error icon and includes a link to the source of the message.

console.error ('Sending an error message to the console.');

.dir()

Displays an interactive list of an object's properties.

console.dir([ 4, false, 'Ian Malcolm' ]);
