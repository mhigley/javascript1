Timers

setTimeout()

The setTimeout() method sets a timer which executes a function or specified piece of code once after the timer expires.

<!-- HTML -->
<button onclick="delayPopup()"> Start Timer</button>
/* JavaScript */
var timer;
function delayPopup() {
timer = setTimeout( popup, 3000);
}
function popup() {
alert('This took a long time to run!') ;
}

On button click, wait 3000 milliseconds then print an alert window to the screen.

clearTimeout()

The clearTimeout() method cancels a timeout previously established by calling setTimeout().

<!-- HTML -->
<button onclick="delayPopup()"> Start Timer</button>
<button onclick="cancelPopup()"> Cancel Timer</button>
/* JavaScript */
var timer;
function delayPopup() {
timer = setTimeout( popup, 3000);
}
function popup() {
alert('This took a long time to run!') ;
}
function cancelPopup( ){
clearTimeout(timer) ;
}

If the Start Timer button has been clicked, Cancel Timer will prevent the popup from printing if pressed within 3000 milliseconds.

setInterval()

The setInterval() method repeatedly calls a function or executes a code snippet, with a fixed time delay between each call.

var foreverGreeting = setInterval (printGreeting, 500);
var greetingNumber = 0 ;
function printGreeting( ){
greetingNumber++;
console.log ('This is greeting number '+greetingNumber);
}

Every 500 milliseconds, increment the greetingNumber and print to the console.

clearInterval()

The clearInterval() method cancels a timeout previously established by calling clearInterval().

var finiteGreeting = setInterval (printGreeting, 500);
var greetingNumber = 0 ;
function printGreeting( ){
greetingNumber++;
if(greetingNumber < 10){
console.log ('This is greeting number '+greetingNumber);
}else{
clearInterval(finiteGreeting );
}
}

Every 500 milliseconds, if the greetingNumber is less than 10, print to the console.
