Targeting DOM Elements

.getElementById();

Returns a reference to the element by its ID; the ID is a string which can be used to uniquely identify the element, found in the HTML id attribute.

<!-- HTML -->
<button id="myBtn" type ="button">Click Me!</ button>
/* JavaScript */
var btn = document .getElementById('myBtn');

.getElementsByTagName();

Returns an array-like object (HTMLCollection) of elements with the given tag name.

<!-- HTML -->
<p> You'd get your first look at the six-foot <a href="#"> turkey</a> as you
move into a clearing. But <a href="#"> raptor</a> , he knew you were
there a long time ago. He moves like a <a href="#"> bird</a> ; lightly,
bobbing his head, And you keep still, because you think
maybe his visual acuity's based on movement, like a < a href="#" >T-
rex</a >, and he'll lose you if you don't move. But no. Not
<a href="#"> VELOCIRAPTOR</a> . You stare at him, and he just stares
back. That's when the attack comes - - not from the
front, no, from the side, from the other two raptors you
didn't even know were there.</p >
/* JavaScript */
var links = document .getElementsByTagName('a');

.getElementsByClassName();

Returns an array-like object (HTMLCollection) of all child elements which have all of the given class names.

Note: This will return a collection of ANY element that has a matching class name! They don't have to be the same tag type.

<!-- HTML -->
<ul>
<li ><a href ="#">Navigation Link One</ a></li >
<li class="dropdown" >
< a href="#" >Navigation Link Two</a >
< ul>
< li><a href="#" >Subnav Link One</a ></li>
< li><a href="#" >Subnav Link Two</a ></li>
< li><a href="#" >Subnav Link Three</a ></li>
</ ul>
</li >
<li class="dropdown" >
< a href="#" >Navigation Link Three</a >
< ul>
< li><a href="#" >Subnav Link One</a ></li>
< li><a href="#" >Subnav Link Two</a ></li>
< li><a href="#" >Subnav Link Three</a ></li>
</ ul>
</li >
<li ><a href ="#">Navigation Link Four</ a></li >
<li class="dropdown" >
< a href="#" >Navigation Link Five</a >
< ul>
< li><a href="#" >Subnav Link One</a ></li>
< li><a href="#" >Subnav Link Two</a ></li>
< li><a href="#" >Subnav Link Three</a ></li>
</ ul>
</li >
</ul>
/* JavaScript */
var dropdownLinks = document .getElementsByClassName('dropdown');

.querySelector();

Returns the first Element within the document that matches the specified selector, or group of selectors. Use of . for classes and # for ID's apply.

/* JavaScript */
// ** using HTML from above example
var oneLink = document .querySelector('ul li:last-child a');

Note: Be cautioned that this selector is for targeting a single item. If multiple exist, it will only return the first of the items.

/* JavaScript */
// ** using HTML from above example
var oneLink = document .querySelector('ul li:last-child li:first-child a');

.querySelectorAll();

Returns a list (NodeList) of the elements within the document that match the specified group of selectors. Use of . for classes and # for ID's apply.

/* JavaScript */
// **using HTML from above example
var allSubLinks = document .querySelectorAll('ul li li a');
