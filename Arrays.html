Arrays

The JavaScript Array object is a global object that is used in the construction of arrays; which are list-like objects. Arrays are used to store multiple values in a single variable. Arrays can also store different types of elements. You can put objects, entire functions or other arrays into arrays!

var colors = [ 'red', 'orange', 'blue'] ;

Arrays are zero based and every item has a corresponding index.

var sodas = [ 'pepsi', 'coke', 'mountain dew' ] ;
console.log (sodas);

Access an Array item

You refer to an array element by referring to the index number.

var soda = sodas [0]; // pepsi
var lastSoda = sodas [sodas.length - 1 ]; // mountain dew

Length property

The length property of an array returns the length of an array (the number of array elements). The length property will always be one higher than the highest index.

sodas.length; // 3

Loop over an Array

sodas.forEach( function(item, index ) {
console.log (index + ' : ' + item );
});
// 0 : pepsi
// 1 : coke
// 2 : mountain dew

Sneak Peek: The forEach() loop is a special type of loop used exclusively for Arrays. It offers a lot of options for accessing the contents of an Array. There are many other types of loops that we'll be exploring soon.

Sneak Peek: Tucked inside of the forEach() is what's called an anonymous function. This function—as shown between the perenthesis—is expecting two things: an item and an index. These "expectations" are called parameters, and are being provided by the forEach() loop. We'll be digging into functions in greater detail soon.

Note: In future examples, sometimes the console.log will be left off for brevity. If the code is to be demonstrated from the external script file, please include console.log with all examples.

Array.isArray()

Determines whether the passed value is an Array.

Array.isArray([ 'red', 'orange', 'green']) ;

.push()

Adds new elements to the end of an array, and returns the new length

var newColorLength = colors .push('violet') ;
console.log (colors);
// ['red', 'orange', 'blue', 'violet']
console.log (newColorLength);
// 4

.pop()

Removes the last element of an array, and returns that element

var lastColor = colors .pop();
console.log (colors);
// ['red', 'orange', 'blue']
console.log (lastColor);
// violet

.unshift()

Adds new elements to the beginning of an array, and returns the new length

var newColorLength = colors .unshift('yellow') ;
console.log (colors);
// ['yellow', 'red', 'orange', 'blue']
console.log (newColorLength);
// 4

.shift()

Removes the first element of an array, and returns that element

var firstColor = colors .shift();
console.log (colors);
// ['red', 'orange', 'blue']
console.log (firstColor);
// yellow

.indexOf()

Search the array for the first instance of an element and returns its position

var colorPosition = colors .indexOf('blue') ; // 2

.lastIndexOf()

Search the array for the first instance of an element, starting at the end, and returns its position

colors.push( 'green', 'violet', 'blue' , 'pink');
console.log (colors);
// ['red', 'orange', 'blue', 'green', 'violet', 'blue', 'pink']
var lastColorPosition = colors .lastIndexOf('blue') ;
console.log (lastColorPosition); // 5

.splice()

Changes the contents of an array by removing existing elements and/or adding new elements.

Remove an item by index position

var removeItem = colors .splice(2, 1 );
console.log (colors);
// ['red', 'orange', 'green', 'violet', 'blue', 'pink']
console.log (removeItem);
// blue

Remove multiple items at an index position

var removeItems = colors .splice(1, 3 );
console.log (colors);
// ['red', 'blue', 'pink']
console.log (removeItems);
// ['orange', 'green', 'violet']

Adding items at index position

var addItems = colors .splice(1, 0 , 'orange', 'yellow', 'green' );
console.log (colors);
// ['red', 'orange', 'yellow', 'green', 'blue', 'pink']
console.log (addItems);
// []

Remove item by index position and add item

var removeAndAdd = colors .splice(5, 1 , 'purple');
console.log (colors);
// ['red', 'orange', 'yellow', 'green', 'blue', 'purple']
console.log (removeAndAdd);
// pink

.slice()

Slice is non-destructive; it does not alter the original array. Elements of the original array are copied into the returned array.

Copying the entirety of an array

var newColors = colors .slice();
console.log (colors);
// ['red', 'orange', 'yellow', 'indigo', 'purple']
console.log (newColors);
// ['red', 'orange', 'yellow', 'indigo', 'purple']

Returning a portion of the original array.

When slicing a portion of an array, note that the second argument is zero-based. It adds up to, but not including the index position.

var newColorsPortion = colors .slice(1, 3 );
console.log (colors);
// ['red', 'orange', 'yellow', 'indigo', 'purple']
console.log (newColorsPortion);
// [orange', 'yellow']

.reverse()

This method reverses an array in place. The first array element becomes the last, and the last array element becomes the first.

var a = [ 'one', 'two', 'three'] ;
var reversed = a .reverse();
console.log (a); // ['three', 'two', 'one']
console.log (reversed); // ['three', 'two', 'one']

.sort()

The method sorts the elements of an array in place and returns the array. The default sort order is according to string Unicode code points.

Default sorting

var points = [ 1, 10, 21 , 2];
points.sort( );
console.log (points);
// [1, 10, 2, 21]
var words = [ 'word', 'Word', '1 Word', '2 Words'];
words.sort( words);
console.log (words);
// ['1 Word', '2 Words', 'Words', 'words']

Tip: In Unicode, numbers come before uppercase letters, which come before lowercase letters. Default sorting typically provides adequate results for alphanumeric sorting.

Comparison sorting

If a comparison function is supplied, the array elements are sorted according to the return value of the compare function. For a simple ascending numerical sort,

var points = [ 4, 2, 5 , 1, 3];
points.sort( function(a, b) {
return a - b ;
});
console.log (points);
// 1, 2, 3, 4, 5

Tip: Comparison sorting typically provides adequate results for numeric sorting.

Here is the breakdown of the calculation:

2, 4, 5, 1, 3 —— 2 is less than 4, move to the next two.
2, 4, 5, 1, 3 —— 4 is less than 5, move to the next two.
2, 4, 5, 1, 3 —— 5 is greater than 1... flip the numbers and work backward.
2, 4, 1, 5, 3 —— 4 is greater than 1... flip the numbers and work backward.
2, 1, 4, 5, 3 —— 2 is greater than 1... flip the numbers. Continue from the index that began the direction shift.
1, 2, 4, 5, 3 —— 5 is greater than 3... flip the numbers and work backward.
1, 2, 4, 3, 5 —— 4 is greater than 3... flip the numbers and work backward.
1, 2, 3, 4, 5 —— 2 is less than 3... sorting complete.
