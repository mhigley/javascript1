XMLHttpRequest()

XMLHttpRequest is an API that provides client functionality for transferring data between a client and a server. It provides an easy way to retrieve data from a URL without having to do a full page refresh. This enables a Web page to update just a part of the page without disrupting what the user is doing.

Everything begins however with a new instance of the XMLHttpRequest().

var req = new XMLHttpRequest();

Methods

.open()

Initializes a request.

req.open( 'GET', 'https://api.github.com');

.send()

Sends the request.

req.send( null);

Properties

.readyState

Holds the status of the XMLHttpRequest. The .readyState is one of two things you want to check before continuing on in your code. The other would be .status;

0: request not initialized
1: server connection established
2: request received
3: processing request
4: request finished and response is ready

req.onload = function(){
if(req. readyState === 4 && req. status === 200){
// code
}
}

.status

Returns the status-number of a request. The .status is one of two things you want to check before continuing on in your code. The other would be .readyState;

200: "OK"
404: "Not Found"
503: "Service Unavailable"
Etc...

req.onload = function(){
if(req. readyState === 4 && req. status === 200){
// code
}
}

.statusText

Returns the status-text (e.g. "OK" or "Not Found")

req.onload = function(){
if(req. readyState === 4 && req. status === 4){
// code
}else{
console.log ('error', req.statusText );
}
}

.responseText

Returns the response data as a string.

req.onload = function(){
if(req. readyState === 4 && req. status === 4){
console.log ('success', req.responseText );
}else{
console.log ('error', req.statusText );
}
}

Complete Example

var req = new XMLHttpRequest();
req.open( 'GET', 'https://api.github.com');
req.onload = function(){
if(req. readyState === 4 && req. status === 4){
console.log ('success', req.responseText );
}else{
console.log ('error', req.statusText );
}
}
req.send( null);

Functionized Example

In modularizing the XMLHttpRequest(), we have an opportunity to dynamically supply the location of the API call and the function that will process the data. See the getReq() parameters that are set. Each time getReq() is called, you can provide a necessary endpoint. For each of these calls, you can provide a function that processes the data specifically for that call.

getReq('https://api.github.com', processAjax);
function getReq(url , callback){
var req = new XMLHttpRequest();
req.open( 'GET', url);
req.onload = function(){
if(req .readyState === 4 && req .status === 4){
callback( JSON.parse(req. responseText));
}else{
console. log('error', req. statusText);
}
}
req.send( null);
}
