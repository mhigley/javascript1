Ternary Operators & Switch Statements

Ternary Operators

This operator is frequently used as a shortcut for the if statement. If the condition is true, the operator returns the value before the colon; otherwise, it returns the value after the colon.

var age = prompt ('How old are you?');
var movieRestriction = age > 18 ?
console.log ('Old enough') :
console.log ('Too young');

It is possible to exceedingly more complicated with the results as well. In the following example, the last comma-separated value of the parenthesis will be the value to be assigned.

// Create two new files in today's project folder: proceed.html & stop.html
// Fill out the html pages with boilerplate and add the following headlines:
// proceed.html: <h1>Welcome to the movies...</h1>
// stop.html: <h1>Sorry, you're not old enough for rated R movies</h1>
var age = prompt ('How old are you?');
var url = age > 18 ?
(
alert('okay, you may proceed.' ), // alert returns undefined, however because it is not the last comma separated value, it is ignored
'proceed.html'
) :
(
alert('sorry, you \'re not quite there yet.'),
'stop.html'
);
window.location. assign(url);

Switch Statements

The switch statement evaluates an expression, matching the expression's value to a case clause, and executes statements associated with that case. The switch expression is evaluated once. If it finds a case that matches, it will excecute the corresponding code.

var today = new Date().getDay ();
switch(today) {
case 0:
case 6:
console.log ('Living for the weekend!');
break;
case 1:
console.log ('Ugh. Mondays');
break;
case 3:
console.log ('Humpday! Halfway there.');
break;
case 5:
console.log ('It\'s Friday, Friday...');
break;
default:
console.log ('Still working.');
}

Note: The break keyword stops any remaining cases from being checked.

Note: the default keyword code will excecute if none of the cases match the expression.
