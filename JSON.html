JSON

JSON, or JavaScript Object Notation, is a minimal, readable format for structuring data. It is used primarily to transmit data between a server and web application.

The two primary parts that make up JSON are keys and values. Together they make a key/value pair.

"characterName" : "Indominus Rex"

characterName is the key, and Indominus Rex is the value.

Any data type can be stored as JSON. You should recognize some of the syntax as covered previously. Arrays use [] and objects use {}.

Arrays

"characters" : ["Alan", "Ellie", "Lex", "Tim", "Ian"];

Objects

"traits" : {
"firstname" : "Matthew",
"lastname" : "Higley",
"eyes" : "green",
"hair" : "brown"
}

Complete Example

{
"user" : {
"first" : "Matthew",
"last" : "Higley"
},
"phone" : "559-252-5555",
"address" : "375 Main St., Fresno CA 93755",
"vehicle" : {
"make" : "jeep",
"model" : "wrangler",
"year" : "2014"
},
"favMovies" : ["Jurassic Park", "Raiders of the Lost Ark", "Holy Grail"]
}
